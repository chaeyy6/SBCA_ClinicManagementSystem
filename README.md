# SBCA Clinic Management System

## Project Overview
The SBCA Clinic Management System is a web-based application for managing **student health records and vaccinations** at San Beda College Alabang (SBCA).  
This project was developed as an academic requirement to improve school clinic processes and streamline record management.

## Objectives
- Digitize student health records for easier access and tracking.
- Maintain accurate vaccination data for students.
- Provide a secure and user-friendly admin interface.
- Ensure data integrity and confidentiality through proper authentication.

## Technologies Used
- **Backend:** PHP
- **Database:** MySQL (via phpMyAdmin)
- **Frontend:** HTML, CSS
- **Tools:** Git, GitHub, VS Code, XAMPP

## Features
- **Student Health Records**  
  - Add, view, update, and delete student health records.  
  - Organized dashboards to track student visits and medical history.

- **Vaccinations**  
  - Record and update student vaccination information.  
  - Proper handling and tracking of vaccination schedules.

- **User Authentication**  
  - Admin login system for secure access.

- **Assets**  
  - Images, icons, and logos used in the application.

- **Config**  
  - Database connection (`db_connect.php`)  
  - Password hashing and security (`generate_hash.php`)
